# Church Management System

A full-stack web application for managing church operations, built with **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**.

## Features

- User registration and login
- Event management (create, update, delete events)
- Donations tracking
- Admin panel for managing members and events
- Fully functional database (SQL file included)

## Technologies Used

- Frontend: HTML, CSS, Bootstrap, JavaScript, JQuery
- Backend: PHP
- Database: MySQL
- Version control: Git/GitHub

## Project Structure
/css → stylesheets
/js → JavaScript files
/images → app images
/sql → database file
index.php → main entry page


## Setup & Installation

1. Clone the repository:

git clone https://github.com/Ubee28/church-management-system.git

2. Import the SQL file in your MySQL database.  
3. Configure database connection in `config.php` (or your config file).  
4. Run the project on a local server (XAMPP, WAMP, etc.).
